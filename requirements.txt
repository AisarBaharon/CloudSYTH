# CloudSynthex Web App - Requirements (Node.js)
# NOTE: For JavaScript projects, this file is informational. Use `bash setup.sh` (recommended)
# or `npm ci` to install exact dependencies from package-lock.json.

Runtime
- Node.js >= 18.17.0 (recommended: 20 LTS)
- npm >= 9 (bundled with Node.js)

System
- Modern web browser (Chrome, Edge, Firefox, Safari)

Install (one-line commands your teammates can run after cloning)

1) Clone the repo (replace <repo-url> with your repository url):

```bash
git clone <repo-url>
cd cloudsynthex
```

2) macOS / Linux (recommended):

```bash
# Optional: use nvm to pick the Node version from .nvmrc
if command -v nvm >/dev/null 2>&1; then
	nvm install
	nvm use
fi

# Run the bundled setup (installs exact deps and generates assets)
bash setup.sh

# Start dev server
npm run dev
```

3) Windows (PowerShell):

```powershell
# From PowerShell (recommended):
powershell -ExecutionPolicy Bypass -File .\setup.ps1

# Or use the cross-platform npm helper (works in cmd/powershell/git-bash):
npm run setup

# Start dev server
npm run dev
```

4) Quick/CI-friendly alternative (exact installs):

```bash
# installs exact versions from package-lock.json
npm ci
# generate icons/assets (postinstall runs this automatically too)
npm run generate:icons
# start dev server
npm run dev
```
